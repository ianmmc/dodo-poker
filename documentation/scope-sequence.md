# Learning Scope & Sequence

**Status: STUB** — Structure defined; content to be developed in dedicated planning session.

---

## Overview

The curriculum covers probability and statistics from zero baseline through advanced decision-making under uncertainty. All content maps to the **Common Core State Standards (CCSS)** mathematics framework, Statistics & Probability strand, grades 6–12.

Progression is **competency-driven**, not age-driven. A student advances by demonstrating understanding, not by completing a fixed number of hands or sessions.

---

## CCSS Standards Targeted

### Middle School (Grades 6–8)
| Standard | Description |
|----------|-------------|
| 7.SP.C.5 | Understand that the probability of a chance event is a number between 0 and 1 |
| 7.SP.C.6 | Approximate probability through repeated experiments; observe long-run frequency |
| 7.SP.C.7a | Develop a uniform probability model; assign equal probability to all outcomes |
| 7.SP.C.7b | Develop a non-uniform probability model; use observed frequencies |
| 7.SP.C.8a | Understand probability of compound events using organized lists, tables, tree diagrams |
| 7.SP.C.8b | Represent sample spaces for compound events |
| 7.SP.C.8c | Design and use simulation to estimate probabilities of compound events |

### High School
| Standard | Description |
|----------|-------------|
| S-CP.A.1 | Describe events as subsets of a sample space; unions, intersections, complements |
| S-CP.A.2 | Understand independence: P(A and B) = P(A)·P(B) |
| S-CP.A.3 | Understand conditional probability; interpret in context |
| S-CP.A.4 | Construct and interpret two-way frequency tables |
| S-CP.A.5 | Recognize independence and conditional probability in everyday language |
| S-CP.B.6 | Find conditional probability using two-way tables or tree diagrams |
| S-CP.B.7 | Apply the Addition Rule: P(A or B) = P(A) + P(B) – P(A and B) |
| S-CP.B.8 | Apply the Multiplication Rule: P(A and B) = P(A)·P(B\|A) |
| S-CP.B.9 | Use permutations and combinations to compute probabilities |
| S-MD.A.1 | Define a random variable; graph its probability distribution |
| S-MD.A.2 | Calculate expected value; interpret in context |
| S-MD.A.3 | Develop a probability distribution from empirical data |
| S-MD.B.5 | Weigh outcomes using probabilities; find expected value; evaluate decisions |
| S-MD.B.6 | Use probabilities to make fair decisions |
| S-MD.B.7 | Analyze decisions using probability concepts |

## Phase Structure

Each phase specifies: learning objectives, CCSS standards, poker variant, NPC(s) introduced, and competency gate criteria.

---

### Phase 1: Foundations of Probability
**Poker Variant:** TBD — likely simplified card-drawing mechanics before full poker  
**NPCs Introduced:** 1 (simple, obvious reasoning pattern)  
**CCSS:** 7.SP.C.5, 7.SP.C.6, 7.SP.C.7a

**Learning Objectives:**
- [ ] Define probability as a number between 0 and 1
- [ ] Enumerate outcomes in a simple sample space (a deck of cards)
- [ ] Calculate simple probabilities (P = favorable / total)
- [ ] Distinguish certain, impossible, and uncertain events
- [ ] Observe that long-run frequency approximates theoretical probability

**NPC:** TBD — embodies complete disregard for probability (acts randomly or always bets)  
**Competency Gate:** TBD

### Phase 2: Compound Events & Counting
**Poker Variant:** TBD — likely five-card draw  
**NPCs Introduced:** +1 (gambler's fallacy or similar)  
**CCSS:** 7.SP.C.8a, 7.SP.C.8b, S-CP.B.9

**Learning Objectives:**
- [ ] Enumerate compound events using tables and tree diagrams
- [ ] Calculate probabilities of compound events
- [ ] Use combinations to count poker hands
- [ ] Distinguish mutually exclusive vs. non-mutually exclusive events

**NPC:** TBD  
**Competency Gate:** TBD

---

### Phase 3: Conditional Probability & Independence
**Poker Variant:** TBD — likely Texas Hold'em introduction  
**NPCs Introduced:** +1–2  
**CCSS:** S-CP.A.1–5, S-CP.B.6–8

**Learning Objectives:**
- [ ] Understand and apply conditional probability P(A|B)
- [ ] Distinguish independent from dependent events
- [ ] Apply the Multiplication Rule
- [ ] Apply the Addition Rule
- [ ] Update probability estimates as new information is revealed (community cards)

**NPC:** TBD  
**Competency Gate:** TBD

---

### Phase 4: Expected Value & Decision-Making
**Poker Variant:** Texas Hold'em  
**NPCs Introduced:** +1–2  
**CCSS:** S-MD.A.1–3, S-MD.B.5–7

**Learning Objectives:**
- [ ] Define and calculate expected value
- [ ] Apply EV to decisions with known probabilities (pot odds)
- [ ] Distinguish short-run variance from long-run expected outcomes
- [ ] Evaluate decisions by expected value, not outcome

**NPC:** TBD  
**Competency Gate:** TBD

---

### Phase 5: Advanced Probabilistic Reasoning
**Poker Variant:** Texas Hold'em (full)  
**NPCs Introduced:** Up to 5 total at the table  
**CCSS:** Full S-CP and S-MD coverage; S-MD.A.3, S-MD.B.5–7

**Learning Objectives:**
- [ ] Reason about opponent hand ranges (probability distributions)
- [ ] Recognize and name cognitive biases in probabilistic reasoning (gambler's fallacy, overconfidence, base rate neglect)
- [ ] Construct and use non-uniform probability models
- [ ] Synthesize all prior concepts in multi-street decision-making

**NPC:** TBD — includes the "rational EV player" as a model of correct reasoning  
**Competency Gate:** TBD
